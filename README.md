# Skver Tashkent — v7.2 World Axis Walk Animation

Эта версия собрана под fullbody-rig модель в T-pose, как на твоём скрине.

## Что изменено

- руки и ноги двигаются через world-axis rotations;
- руки больше не должны махать вверх/вниз;
- ноги должны делать шаги вперёд/назад;
- добавлена регулировка силы шага:
  - `[` уменьшить амплитуду;
  - `]` увеличить амплитуду;
- отдельные файлы анимации не нужны.

## Как использовать

Положи модель сюда:

```text
public/assets/models/player.glb
```

Запуск:

```bash
npm install
npm start
```

Открыть:

```text
http://localhost:3000
```

После запуска нажми `Ctrl + F5`.

## Важно

Это всё ещё procedural-анимация. Самый качественный вариант — настоящие motion-captured анимации, но для твоей ситуации без отдельных файлов это лучший вариант.
